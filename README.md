=======================

Find quickly a number between 1 and 100.

#### How to install?
	doas bundle install # or sudo bundle install (depending your OS)

#### How to run the game?
	./bin/game

Testing
-------

#### Execute rspec (spec/core_spec.rb)
	rake

#### Test your Ruby style programming with rubocop
	rake style

#### Run rspec/rubocop
	rake all

#### Get this help
	rake -T
